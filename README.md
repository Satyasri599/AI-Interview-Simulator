# 🤖 AI-Driven Interview Simulation Platform
**Mini Project | Backend Development & System Logic**

### 🌟 Project Overview
The AI-Driven Interview Simulation Platform is a high-fidelity recruitment tool designed to automate and streamline the initial stages of technical screening. By simulating a real-world professional atmosphere, the platform evaluates candidates through five specialized assessment rounds: **Aptitude, Technical, Theory, Managerial, and HR**.

---

### 🚀 Key Implementation & Architecture

* **End-to-End User Journey:** Engineered a seamless workflow that transitions candidates from secure authentication and resume processing into a structured multi-round environment.
* **Stateful Session Management:** Leveraged **Flask-Sessions** to track performance data in real-time, ensuring data integrity and persistence across all five assessment modules.
* **Dynamic Content Engine:** Developed a decoupled **JSON-based backend** architecture. This allows for scalable question bank management and dynamic content delivery without modifying the core application logic.
* **Interactive Coding Simulation:** Integrated an **AJAX-powered** coding interface to provide a low-latency, responsive IDE experience for technical evaluations.
* **Automated Evaluation System:** Architected a custom scoring algorithm that generates weighted results and AI-driven feedback, categorizing performance into actionable metrics.
* **Professional UI/UX:** Designed a responsive interface featuring a "Robot Assistant" and modern CSS animations to simulate a formal interview environment.

---

### 🛠️ Technical Stack
* **Backend Framework:** Python (Flask), Jinja2
* **Frontend Technologies:** HTML5, CSS3 (Bootstrap), JavaScript (AJAX)
* **Data Management:** JSON-based persistent storage
* **Environment:** Developed and debugged using VS Code & Git
